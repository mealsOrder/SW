Title: To '' vs ""
==============================================================

1. 문자 // 그 문자에 해당하는 아스키 코드가 튀어나옴
	character
	

2. 문자열 // 그 문자열에 해당하는 주소가 튀어나옴
	string
	
	'A' vs "A" 무슨 차이 일까?
	
	printf("%d", 'A') // 65
	printf("%c", 'A') // A
	
	"--" : --의 주소값을 리턴함
	printf("%d", "A") // 주소 값을 정수형으로 출력
	printf("%c", "A") // 주소 값에서 한글자만 출력 (재수 좋으면 0-9a-Z 출력됨)
	printf("%s", "A") // A , null 이 나올때 까지 문자열을 출력 
	
	우리는 4바이트 주소 체계 사용한다~
	그때 gcc -m hello.c 이렇게 컴파일 해주면 됨
