Title: To pointer array equation
==============================================================

"포인터 배열 등가식"
A[N] == *(A+N);
ex.
	a[3] == *(a+3);

#include<stdio.h>

int main(){
	
	int a[5] = {10,1,2,3,4};
	
	printf("%d",&a[0]); // 1000
	-> 배열 a의 첫번째 (0) 요소의 주소 
	
	printf("%d",a); 	// 1000
	-> 배열 a의 첫번째 (0) 요소의 주소
	
	printf("%d",&a); 	// 1000
	-> 배열 a의 시작 주소
	
	return 0;
}

a[0] == *(a+0) == *(a) == *a
a[0] == *a
양변에 & 취해주고
&a[0] == &*a
&a[0] == a

그래서 배열의 이름은 배열 첫번째 요소의 주소이다!
