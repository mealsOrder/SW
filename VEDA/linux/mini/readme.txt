1. 일반 버전
server.c s , client.c 를 c로 컴파일 해서
./s 로 서버를 돌리고
./c 127.0.0.1 로 접속하여 여러 클라이언트들을 실행하여 
채팅하고 각 클라이언트들의 유저 이름 으로 각 유저가 작성한 메시지를 파일에 저장
검색 메뉴에서 내가 보고싶은 유저의 이름을 입력하면 해당 유저가 작성한 메시지를 시간 순으로 보여줌

파일 검색은 dir 파일의 ./c 127.0.0.1 로 돌려서 전송하면 해당 클라이언트가 있는 곳으로 파일을 전송한다.
파일은 test.txt 이고 내용이 있는데 확인해보면 잘 전송된 것을 확인할 수 있다.

2. UI 버전
서버는 그대로 ./s 로 서버를 돌리고
ui.c 를 u로 컴파일 해서 ./u 127.0.0.1 로 클라이언트를 실행

ppt에는 잘 되는 부분을 캡처해서 올려 놓았음
(버퍼 문제, 커서이동 문제, 메시지 출력 문제, 화면 지움 문제)등 문제가 생각보다 많다...
ui를 사용하면 문제가 많지만 해당 라이브러리를 공부하면서 ui를 구현 해보았다는것에 의미를 두고 
조금이나마 점수를 받으려고 포함시켰음

사실 시간이 좀더 많으면 더 잘 구현해서 멋진 ui로 채팅프로그램을 만들고 싶었습니다.

추석 기간동안 마무리 잘해서 멋진 프로그램으로 완성해 오겠습니다. 이상입니다.
