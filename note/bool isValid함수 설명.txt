bool isValid(int row) {
    for (int i = 0; i < row; i++) {
        // 같은 열에 퀸이 있는 경우
        if (board[i] == board[row])
            return false;
        // 대각선 위 또는 대각선 아래에 퀸이 있는 경우
        if (abs(board[i] - board[row]) == row - i)
            return false;
    }
    // 충돌이 없는 경우 true 반환
    return true;
}



for 문을 통해 현재 행(row) 이전의 모든 행(0 ~ row-1)을 확인합니다.
첫 번째 if 문에서는 같은 열에 퀸이 있는지 확인합니다.
board[i]는 i번째 행의 퀸의 열 위치를 나타내고, board[row]는 현재 행의 퀸의 열 위치를 나타냅니다.
이 둘이 같다면 같은 열에 퀸이 있다는 뜻이므로 false를 반환합니다.
두 번째 if 문에서는 대각선 위 또는 대각선 아래에 퀸이 있는지 확인합니다.
abs(board[i] - board[row]) 는 i번째 행의 퀸과 현재 행의 퀸 사이의 열 차이의 절댓값을 나타냅니다.
row - i는 현재 행과 i번째 행 사이의 행 차이를 나타냅니다.
이 두 값이 같다면 대각선 상에 퀸이 있다는 뜻이므로 false를 반환합니다.
위 두 가지 검사를 통과하면 현재 행의 퀸이 이전 행들의 퀸들과 충돌하지 않는다는 뜻이므로 true를 반환합니다.
왜 앞에는 abs()가 있고 뒤에는 없을까요?

앞의 abs(board[i] - board[row])는 열 차이의 절대값을 구하는 것입니다. 이는 대각선 상에 퀸이 있는지 확인하기 위해 필요한데, 대각선 상에 있는 경우 열 차이의 절대값이 행 차이와 같기 때문입니다.
반면, row - i는 행 차이를 나타내는 것이므로 절대값을 구할 필요가 없습니다.
이렇게 isValid() 함수에서는 현재 행의 퀸이 이전 행들의 퀸들과 같은 열 또는 대각선 상에 있는지 확인하여 유효성을 검사하고 있습니다. 궁금한 점이 더 있다면 말씀해주세요.
