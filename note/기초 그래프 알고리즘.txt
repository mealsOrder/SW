기초 그래프 알고리즘

- 최단경로 알고리즘
- Strongly Connected Component
- 최소 신장 트리

최단경로 알고리즘
- 다익스트라 알고리즘 : 하나의 정점에서 모든 정점까지 최단거리를 구하는 알고리즘 (간선 가중치가 양수일때만)
- 폴로이드 알고리즘 : 모든 점점에서 모든 정점까지 최단거리를 구하는 알고리즘
- 벨만포드 알고리즘 : 하나의 정점에서 모든 정점까지 최단거리를 구하는 알고리즘 (양수,음수 모두 가능)

최단경로의 특징
- 정점 x까지 최단거리로 가기 위해서는 그 직전까지도 최단거리로 가야한다!
- 인접해 있는 정점을 비교해서 그중 가중치가 가장 작은것을 선택한다!

다익스트라 알고리즘
- 한 점에서 모든 점까지의 최단거리를 구함
- 그것은 최단경로 트리(Shortest-path Tree)가 된다
  Tree : 사이클이 없는 그래프

- 그래서 최단 경로 트리를 어떻게 만들 것인가?
   T(i) = i까지 도달하는 최단거리
   각 정점의 초기 거리 값 INF
   (1) 색칠 안한 것 중 최소값 찾기
   (2) 색칠
   (3) 뻗는다
