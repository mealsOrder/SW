# 자리배치 프로그램
랜덤하게 자리를 배치하는 프로그램 ver1.0

# 목적
VEDA 사람들 마다 고유의 번호가 있는데
자리를 공평하게 랜덤하게 정하기 위해서 만들었음

# 기능
- 자리 랜덤 배치
- 벡터에 고유의 번호를 하나씩 넣어 셔플로 랜덤하게 섞고
  2차원 벡터에 섞은 값을 지정 한다.

# 사용법
- 총인원, 세로, 가로 입력
- 칠판을 기준으로 자리 배치
